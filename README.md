# Final_Fantasy_2_Composer
WIP FF2J Sound composer

made to edit ff2 NES Sound data

========================================

Numpad 1 - 6 --> Octaves 2 - 7

Numpad 7 - 9 --> Whole/Half/4th Note

Numpad / and * --> Shift current Key up / down

Numpad + --> Add note

Numpad - --> Add Rest

you can also just use the mouse.
